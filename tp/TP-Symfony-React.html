1. Installer et prendre en main Symfony et EasyCodingStandard

Via composer créer un nouveau projet avec la dernière version stable de notre framework préféré.

Créer le projet dans le répertoire /var/www/my_project du conteneur PHP7.

Vérifier que tous les prérequis sont présents pour pouvoir coder le projet Symfony (en ligne de commande et en navigation web).

Accéder au front avec les URLs http://localhost/my_project/web/ ou http://localhost/my_project/web/app_dev.php

Modifier le fichier app_dev.php pour pouvoir y accéder.

Mettre en place Symplify/EasyCodingStandard pour respecter les conventions de code Symfony.

Créer votre première page “Hello World!” pour afficher Hello votre nom.


Solution
Se connecter au conteneur PHP7
docker exec -ti --user www-data php7 bash

Exécuter ces commandes pour créer le projet dans /var/www/my_project

www-data@php7:~/frontend$ cd
www-data@php7:~$: composer create-project symfony/skeleton my_project

Vérifier les prerequis en CLI

www-data@php7:~/my_project$: composer req requirements-checker
www-data@php7:~/my_project$: php vendor/bin/requirements-checker

En navigation web http://localhost/my_project_lts/public/check.php

www-data@php7:~/my_project$: composer remove requirements-checker

Mise en place de Symplify/EasyCodingStandard

www-data@php7:~/my_project$: composer require --dev symplify/easy-coding-standard
www-data@php7:~/my_project$: touch easy-coding-standard.neon


#easy-coding-standard.neon
includes:
    - vendor/symplify/easy-coding-standard/config/symfony.neon

<?php

namespace App\Form;

use Symfony\Component\Form\AbstractType;
use Symfony\Component\Form\FormBuilderInterface;
use Symfony\Component\Form\Extension\Core\Type\TextType;
use Symfony\Component\Form\Extension\Core\Type\EmailType;
use Symfony\Component\Form\Extension\Core\Type\TextareaType;
use Symfony\Component\Form\Extension\Core\Type\BirthdayType;
use Symfony\Component\Form\Extension\Core\Type\SubmitType;
use Symfony\Component\Validator\Constraints\Length;
use Symfony\Component\Validator\Constraints\NotBlank;

class ContactType extends AbstractType
{
    public function buildForm(FormBuilderInterface $builder, array $options)
    {
        $builder
            ->add(
                'name',
                TextType::class,
                [
                'constraints' => [
                  new NotBlank(),
                  new Length(array('min' => 1, 'max' => 100)),
                ],
                ]
            )
            ->add('email', EmailType::class)
            ->add('message', TextareaType::class)
            ->add('birthday', BirthdayType::class)
            ->add('send', SubmitType::class);
    }
}

Créer un service dédié à l’envoi d’email.

Le contenu des emails est rendu avec twig et le message sera envoyé avec swiftmailer.

<?php
/**

namespace App\Service;

class EmailManager
{
  private $twig;
  private $mailer;

  public function __construct(\Twig_Environment $twig, \Swift_Mailer $mailer)
  {
    $this->twig = $twig;
    $this->mailer = $mailer;
  }

  public function sendRegistrationEmail($email)
  {
    $message = (new \Swift_Message('Hello Email'))
      ->setFrom('send@example.com')
      ->setTo($email)
      ->setBody(
        $this->twig->render(
        // src/Resources/views/Emails/registration.html.twig
          'Emails/registration.html.twig',
          ['email' => $email]
        ),
        'text/html'
      );
    return $this->mailer->send($message);
  }

}
